# Annual Legal Audit Checklist

**Owner**: Lumenshore Limited  
**Schedule**: January each year  
**Next audit due**: January 2027

---

## 1. Privacy Policy Accuracy

- [ ] Review actual data collection — does Privacy Policy match?
- [ ] Check all listed third-party services are still in use (Sentry, Vercel Analytics)
- [ ] Verify data flow descriptions are current
- [ ] Confirm data retention periods are accurate
- [ ] Review cookie/tracking disclosures against real implementation
- [ ] Check GPC/DNT signal handling is correctly described

## 2. Terms of Service Accuracy

- [ ] Verify subscription tiers match current App Store offerings
- [ ] Review liability caps against current pricing
- [ ] Check geographic restrictions are current
- [ ] Confirm dispute resolution mechanisms are valid
- [ ] Verify company details (registration number, VAT, address)
- [ ] Review consumer rights disclosures for UK, EU, US, Australia

## 3. Cookie Policy Accuracy

- [ ] Audit actual cookies set by lumenlingo.com
- [ ] Check third-party scripts (analytics, error tracking)
- [ ] Verify consent mechanism is functioning
- [ ] Review cookie lifetimes against declared durations

## 4. App Store Compliance

- [ ] Review latest Apple App Store Review Guidelines
- [ ] Check for new WWDC policy changes
- [ ] Verify App Privacy Nutrition Label accuracy
- [ ] Confirm EULA / terms link in App Store Connect is current

## 5. GDPR Compliance

- [ ] Check for new EDPB (European Data Protection Board) guidance
- [ ] Review DPA (Data Protection Authority) enforcement actions relevant to apps
- [ ] Assess whether GDPR Art. 27 EU representative appointment is now required
- [ ] Verify DPIA (Data Protection Impact Assessment) is up to date if applicable
- [ ] Review lawful basis for processing against current features

## 6. CCPA / US State Law Compliance

- [ ] Check for newly enacted US state privacy laws
- [ ] Review California AG / CPPA enforcement updates
- [ ] Verify "Do Not Sell or Share" disclosures
- [ ] Assess Virginia, Colorado, Connecticut, Utah, Texas, Oregon, Montana law requirements
- [ ] Review any new state-specific opt-out requirements

## 7. UK Compliance

- [ ] Check for UK Data Reform Bill / DPDI Bill updates
- [ ] Review ICO guidance updates
- [ ] Verify ICO registration is current and renewed
- [ ] Review UK-specific consumer protection requirements
- [ ] Check UK GDPR divergence from EU GDPR

## 8. DSA / DMA Compliance

- [ ] Review Digital Services Act enforcement developments
- [ ] Check Digital Markets Act changes affecting app distribution
- [ ] Reassess intermediary service classification if UGC features added
- [ ] Verify DSA transparency information is current

## 9. Insurance & Risk

- [ ] Review professional indemnity / cyber insurance coverage
- [ ] Assess coverage adequacy against current revenue/user base
- [ ] Check policy renewal dates

## 10. Emerging Regulations

- [ ] EU AI Act — assess applicability if AI features added
- [ ] EU Data Act — review data portability obligations
- [ ] India DPDPA — assess if targeting Indian users
- [ ] UK Online Safety Act — review if UGC features planned
- [ ] Children's privacy (COPPA, UK Age Appropriate Design Code) — assess if any features attract under-13s
