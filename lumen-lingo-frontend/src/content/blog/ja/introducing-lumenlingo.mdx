言語学習はほとんどの人にとって苦行のように感じられます — 無限のフラッシュカードを、蛍光灯の下で無機質なインターフェースを使って繰り返す日々。LumenLingoを作ったのは、こんな疑問からでした：**学習体験そのものが美しく、穏やかで、実際に楽しみたくなるものだったらどうだろう？**

## なぜ没入型なのか

認知科学は明確な事実を示しています：学習環境の**コンテキスト**は、学習内容そのものと同じくらい記憶の定着に影響を与えます。静かだが味気ない環境で学んだ単語よりも、豊かな感覚体験の中で学んだ単語の方が、はるかに記憶に残りやすいのです。

LumenLingoはこの原則に基づいて設計されています：

- **コズミック背景** — 宇宙をテーマにした動的な背景が、独自の視覚的コンテキストを作り出します
- **アンビエントサウンドスケープ** — カスタムデザインの音風景が、副交感神経を活性化し、集中した学習状態を促進します
- **カームデザイン** — ミニマリストのインターフェースが認知負荷を最小限に抑え、語彙に集中できるようにします

<Callout variant="info">
マイヤー（2009年）の研究では、視覚と聴覚のチャンネルを組み合わせたマルチメディア提示が、単一チャンネルの提示よりも一貫して優れた学習成果を生むことが示されています。
</Callout>

## 科学に基づく間隔反復

LumenLingoの核心は**適応型間隔反復アルゴリズム**です。すべてのカードに一律のスケジュールを適用するのではなく、あなた個人のパフォーマンスに基づいて最適な復習タイミングを計算します。

<FeatureHighlight title="スマートスケジューリング" icon="🧠">
アルゴリズムは正解・不正解だけでなく、回答速度も追跡します。800msでの正解は強い記憶を示し、4000msでの正解は弱い記憶を示します。間隔はそれに応じて調整されます。
</FeatureHighlight>

LumenLingoは語彙学習を真摯に受け止めるすべての学習者のために設計されています：

| 機能 | 説明 |

| 適応型SRS | あなたの学習パターンに適応するアルゴリズム |
| サウンドスケープ | 集中学習のための没入型アンビエント音楽 |
| コズミック背景 | 動的でカスタマイズ可能な宇宙テーマビジュアル |
| ダークファースト設計 | 夜間学習セッションに最適化 |
| ネイティブ発音 | すべてのカードにネイティブ音声付き |
| 呼吸エクササイズ | 集中力を高めるガイド付き呼吸法 |
