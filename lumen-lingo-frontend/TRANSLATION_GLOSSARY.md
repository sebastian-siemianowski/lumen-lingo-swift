## 5. Rules

1. **Never translate brand names** — LumenLingo, LumenShore, App Store always appear in Roman script.
2. **Tier names stay in English** — Free, Pro, Elite, Royal are universal marketing labels.
3. **Feature names are adapted** — Use natural, evocative equivalents rather than literal translations.
4. **CJK languages** use katakana (JA) or established Chinese terms (ZH) for English-origin feature names where a natural translation doesn't exist.
5. **Arabic** translations should feel formal but approachable (MSA register).
6. **Consistency** — If a term is established in a locale's message file, all future content must use the same translation.
